Para ejecutar este proyecto, el usuario necesitará en su computadora 
las siguientes herramientas:
-Python
-Flask

Se puede instalar python a través de su página oficial:
https://www.python.org/downloads/

Una vez que se tenga Python, para descargar flask se debe de ejecutar 
el siguiente comando: pip install flask

Para correr el proyecto, dentro de la carpeta, se deben de ejecutar los
siguientes comandos en la terminal:
    export FLASK_APP=app.py
    flask run 

Al ejecutarse, la terminal indicará el sitio en la computadora desde donde el programa
está corriendo. 
Se deberá de acceder a esta direccion a través de los siguentes navegadores:
-Firefox
-Safari

El programa debería de funcionar sin problemas.
